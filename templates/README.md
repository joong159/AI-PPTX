# PPT 템플릿 폴더

이 폴더에 `.pptx` 파일을 넣으면 앱이 자동으로 인식합니다.

## 무료 템플릿 다운로드 사이트

| 사이트 | 특징 |
|--------|------|
| https://slidescarnival.com | 깔끔하고 다양한 무료 테마 |
| https://slidebazaar.com | 비즈니스 스타일 다수 |
| https://slidesmania.com | 창의적이고 모던한 디자인 |
| https://canva.com | 디자인 후 PPTX 내보내기 가능 |
| https://presentationgo.com | 다이어그램 포함 템플릿 |

## 사용 방법

1. 위 사이트에서 마음에 드는 템플릿을 `.pptx` 형식으로 다운로드
2. 이 `templates/` 폴더에 복사
3. 파일명이 앱에서 템플릿 이름으로 표시됨 (예: `corporate_blue.pptx` → "Corporate Blue")
4. 앱 Step 2에서 해당 템플릿 선택

## 주의사항

- 파일명에 한글보다 영문/숫자 권장 (예: `modern_clean.pptx`)
- 파일 열려 있는 상태에서 앱 사용 시 충돌 가능 (파일 닫고 사용)
- 템플릿의 "슬라이드 마스터" 디자인이 최종 PPTX에 적용됨
